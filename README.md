# 💇 Sistema Salão Serviços

Aplicação desktop desenvolvida em Java com o objetivo de gerenciar clientes, tipos de serviços e serviços prestados em um salão de corte de cabelo, utilizando interface gráfica com Swing e banco de dados MySQL.

## 🛠 Tecnologias

* Java
* Java Swing
* Java AWT
* MySQL
* NetBeans

## 📌 Funcionalidades

* Cadastro de clientes
* Cadastro de tipos de serviços
* Cadastro de serviços prestados
* Consulta de registros
* Alteração de registros
* Relacionamento entre tabelas
* Interface gráfica com Swing

## 🚀 Projeto

Projeto desenvolvido em grupo como parte avaliativa da nota P2, com foco na aplicação de conceitos de Programação Orientada a Objetos, interfaces gráficas em Java e persistência de dados utilizando MySQL.
